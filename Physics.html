    What is physics 1️⃣ Frontend (Client Side) This is what users see and interact with. Examples: • Pages • Buttons • Videos • Forms • Navigation Technologies: • HTML • CSS • JavaScript • Frameworks like React, Vue, Angular Example for a video site: • Video player • Comments section • Like button • Search bar Frontend sends requests to the backend to get or send data. ________________________________________ 2️⃣ Backend (Server Side) The backend is the brain of the website. It: • Processes requests • Handles logic • Communicates with databases • Sends responses to frontend Example tasks: • Login authentication • Upload videos • Process payments • Recommend content Technologies: • Python (Django, Flask) • Node.js • Java • Go • Ruby Example: When you click “Subscribe” on YouTube, the backend records it. ________________________________________ 3️⃣ Database The memory of the website. Stores: • User accounts • Videos • Comments • Likes • Payments Common databases: • PostgreSQL • MySQL • MongoDB • Redis Example database tables: Users Videos Comments Subscriptions Without a database, the site cannot remember anything. ________________________________________ 4️⃣ APIs APIs allow frontend and backend to communicate. Example request: Frontend → “Give me the latest videos” Backend → returns JSON data Example API response: { "title": "Learn Python", "views": 10000 } APIs also allow mobile apps to use the same backend
